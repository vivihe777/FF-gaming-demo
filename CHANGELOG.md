# Changelog

## 2026-05-18

- 发布 v.0.18 本地体验版：
  - 主页面左下角增加版本号 `v.0.18`，游戏开始后自动隐藏。
  - 飞跃 20 个障碍后开启天气系统，随机出现闪电、暴雨、龙卷风、雷暴等视觉效果。
  - 更新静态 harness 校验，覆盖版本展示和天气系统关键入口。
- 修复 CloudBase 静态资源缓存：
  - 为 CSS 和主脚本增加版本参数，避免腾讯云 CDN 或浏览器继续使用旧的 `src/main.js`。
  - 解决 CloudBase 更新后 PC 端仍看到旧背景横线的问题。
- 发布 PC 与移动端体验优化版：
  - 去除背景横向纹理和后景云层移动，减少桌面端视觉干扰与背景绘制开销。
  - 将空格键改为全局输入监听，修复页面焦点变化后空格无法起飞的问题。
  - 移动端按设备像素比提升画布分辨率，并收掉 HUD 与菜单的高成本模糊效果，提升文字和画面清晰度。
  - 点击起飞继续沿用轻量状态插值和拖尾对象池，降低每次点击的瞬时卡顿。
- 修复 CloudBase 子路径部署资源加载：
  - 将 CSS、Phaser 和主脚本引用改为相对路径，兼容 `/FF-gaming-demo/` 这类部署路径。
  - 解决腾讯云页面只显示原生 HTML、样式和游戏画布不加载的问题。
- 增加腾讯云 CloudBase 迁移支持：
  - 新增通用静态构建命令 `npm run build:static`，输出目录为 `dist/`。
  - 保留 Vercel 专用构建命令不变，支持 Vercel 和 CloudBase 双平台部署。
- 发布点击性能优化版：
  - 移除点击起飞时即时创建的缩放 tween，改为轻量状态插值，降低点击瞬间卡顿。
  - 拖尾特效改为对象池复用，避免每帧创建和销毁圆点导致网页端掉帧。
  - 护盾描边只在状态变化时更新，减少每帧重复样式写入。
- 发布关卡路线合理性修正版：
  - 重排 10 关速度、门距、重力和风力曲线，降低前期突兀难度。
  - 限制相邻风门中心高度差，避免生成玩家物理上飞不过去的路线。
  - 保留后期递进挑战，但让每关难度更平滑。
- 发布爬升能力修正版：
  - 保持短平快节奏不变，增强单次点击的瞬间上抬速度。
  - 改善向高处飞行需要过度连点的问题，同时不恢复额外持续升力。
- 发布短平快节奏调优版：
  - 将单次点击后的持续飞跃时间压短，取消额外持续升力。
  - 提高上升阶段回落重力，让小鸟更早低头下坠，形成更明显的节奏顿挫。
  - 收短点击后的前进加速窗口，避免一次点击带来过长滑行。
- 发布飞行手感调优版：
  - 强化点击响应：起飞速度更轻盈，短时间连续点击会有更敏捷的抬升反馈。
  - 调整飞行动力：降低单次上升幅度，下坠阶段使用更强的加速度和俯冲姿态，和上升弧线形成明显差别。
  - 收敛俯冲姿态，并在点击后加入短暂前进加速，让单次起飞更有推进感。
  - 强化下坠中的抬头恢复：点击后不只稳住下落，还会获得少量向上飞行和前进推进。
  - 缩短上升后的滞空滑行，点击瞬间更快抬头，并更快进入低头下坠。
  - 去掉风门内侧旋转星星装饰，减少穿越时的视觉干扰。
- 发布初始版 Flappy Quest：10 关闯关、3 套皮肤、奖励机制、暂停/重开/主界面。
- 修复 Vercel 部署 404：构建脚本生成标准 `.vercel/output` 静态产物。
- 更新视觉与手感：
  - 风门障碍物增加灯带、边框、铆钉、顶帽和旋转装饰。
  - 每 2 关切换一个背景主题，共 5 组环境氛围。
  - 小鸟上升和下坠采用不同重力，下坠更快且设定终端速度。
